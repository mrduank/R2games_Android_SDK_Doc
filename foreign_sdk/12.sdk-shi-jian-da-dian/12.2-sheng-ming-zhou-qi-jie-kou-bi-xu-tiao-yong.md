# 12.2生命周期接口（必须调用）

请在游戏**主Activity**的生命周期方法onResume和onPause方法中， 添加如下代码：

```text
@Override
protected void onResume(){
    super.onResume();
    R2TrackApi.getInstance().onResume(this,new TrackEvent());
}
@Override
public void onPause(){
    super.onPause();
    R2TrackApi.getInstance().onPause(this,new TrackEvent());
}

```
