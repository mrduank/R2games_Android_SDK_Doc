# 3.1 SDK初始化接口（必须接入）



**功能说明：**

研发方需在游戏打开后调用该接口一次且仅仅一次，可在游戏自定义`Application`类中的`onCreate()`方法中调用该接口。

**接口形式：**

```text
public void initApp()
```

**接口示例：**

```text
public class GameApplication extends Application {
    @Override
    protected void onCreate() {
        R2SDK.initApp (this);
    }
}
```
