# 3.7 展示广告图片接口（按需求接入）

**功能说明：**

研发方可以调用该接口实现在游戏中展示该游戏的广告图片。

**接口形式：**

```text
public void showAdImage(Activity act, R2AdCallback callback)
```

**参数说明：**  
 act : Activity实例（必须）

callback : 接口回调（必须）

**接口示例：**

```text
R2SDK.showAdImage(this, new R2AdCallback() {
    @Override
    public void onCancel() {
    }
});
```
