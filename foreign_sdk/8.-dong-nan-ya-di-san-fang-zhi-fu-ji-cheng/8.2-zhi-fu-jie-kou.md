# 8.1支付接口

**接口说明：**

研发可以调用该接口在客户端打开第三方东南亚支付界面,让用户完成第三方支付。

**接口形式及说明如下:**

```text
/*
userId : 当前账号的r2 uid,由SDK登录接口返回
productId : 当前支付产品对应的产品ID,由SDK服务器技术端分配
productDisplayName :产品的显示名称用于客户端UI显示,可选参数可传空字符””
roleId : 当前玩家角色唯一标示ID
serverId : 当前所处游戏服务器唯一标示ID
extraData : 游戏方自定义数据,SDK服务器通知游戏服务器发货的时候原样返回
gameCno : 游戏研发方自身的订单号,务必保证其唯一性
payRegion : 支付地区代码(香港传HK  台湾传TW  新加坡传SG 马来西亚传MY  越南传VN等等)
payChannel : 第三方支付厂商名称,由SDK服务器端定义(MyCard传mycard,UPay传upay等等)
    */
    public static void startTppPaySA (Context context,
                           String userId,
                           String productId,
                           String productDisplayName,
                           String roleId,
                           String serverId,
                           String extraData,
                           String gameCno,
                           String payRegion,
                           String payChannel,
                           R2Callback<String> callback)

```



**接口示例：**

```text
       String userId = "30000001";
       String productId = "3644";
       String productDisplayName = "新手首冲礼包";
       String roleId = "uid1458965";
       String serverId = "s2";
       String extraData = "sdktesting";
       String gameCno = "gcno" + System.currentTimeMillis();
       String payRegion = "TW";
       String payChannel = "mycard";
  R2SDK.startTppPaySA (this.getBaseContext(),
                            userId,
                            productId,
                            productDisplayName,
                            roleId,
                            serverId,
                            extraData,
                            gameCno,
                            payRegion,
                            payChannel,
                            new R2Callback<String>() {
                @Override
                public void onSuccess(String result) {
                   //支付成功
                    Log.i("r2_sdk","finally,pay successfully");
                }
                @Override
                public void onCancel() {
                    //支付取消
                    Log.e("r2_sdk","finally,cancelled");
                }
                @Override
                public void onError(R2Error error) {
                    //支付失败
                    Log.e("r2_sdk","finally,failed to pay, err => " + error.toString());
                }
        });
```
